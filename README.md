# Baddies Writes Blog App

A backend-focused blog application built using Node.js, Express, and EJS that allows users to create, read, and manage blog posts.

---

## 🚀 Features

* Create new blog posts
* View all posts
* View individual post details
* Edit and delete posts *(if implemented)*
* Server-side rendering using EJS

---

## 🛠 Tech Stack

* Node.js
* Express.js
* EJS (templating)
* Basic CSS

---

## 📦 Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Aditityagivats/baddies-writes-blog-app.git
```

2. Navigate into the project

```bash
cd baddies-writes-blog-app
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
npm start
```

Or for development (if using nodemon):

```bash
npm run dev
```
